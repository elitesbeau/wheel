# Market Elites - Wheel Strategy Platform

A modern web application for options traders using the wheel strategy. Includes both user-facing trading tools and comprehensive admin dashboard.

## Features

### User Application (`index.html`)
- Cash Secured Put (CSP) & Covered Call (CC) calculator
- Portfolio tracking and position management
- Real-time trading signals
- Alpaca Markets API integration
- AI assistant (Wheely) for strategy guidance
- Educational resources for wheel strategy
- Dark/Light theme support

### Admin Dashboard (`admin.html`)
- User management and analytics
- Trading signal creation and management
- Revenue and subscription tracking
- Platform settings and configuration
- Database management tools
